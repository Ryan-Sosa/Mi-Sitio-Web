ACERCA DE 3D - TECNO

3D - TECNO es un sitio dedicado a explicar de manera sencilla los componentes del hardware y el software.

El objetivo del sitio web es ayudar a comprender el funcionamiento de su computadora, desde los componentes físicos, hasta los programas que son utilizados diariamente.

El sitio está pensado como una guía para aquellos que quieren aprender un poco de tecnología de manera accesible.

El sitio web fue desarrollado por: Ryan Rodrigo Sosa.
